CV Dashboard Pipeline

An image-to-dashboard computer vision pipeline that converts screenshots into interactive analytics.

Overview

This project was built at Strategy to help businesses extract structured data from dashboard screenshots. Users upload an image of a chart or dashboard, and the system automatically identifies the visualization type, extracts data points, and generates an interactive version.

Tech Stack

  • Computer Vision: Custom PyTorch models for chart classification
  • OCR: Tesseract + custom post-processing for data extraction
  • Data Processing: pandas for structured data manipulation
  • Visualization: Plotly for interactive chart generation
  • Backend: FastAPI with async processing
  • Infrastructure: AWS Lambda + S3 for scalable processing

Key Features

  • Chart classification: Automatically detects bar, line, pie, and scatter charts
  • Data extraction: Pulls numerical values and labels from images
  • Interactive output: Generates zoomable, hoverable Plotly charts
  • Batch processing: Handles multiple images in parallel
  • Export formats: CSV, JSON, and embeddable HTML

Challenges & Solutions

The main challenges were handling low-quality screenshots and dealing with the variety of chart styles. I addressed these by:

  • Training on a diverse dataset of real-world dashboard screenshots
  • Implementing robust image preprocessing (denoising, contrast enhancement)
  • Building confidence scoring to flag uncertain extractions for human review

Results

The pipeline achieved 92% accuracy on chart classification and 87% accuracy on data extraction. It's now used by multiple enterprise clients to digitize legacy reports and dashboards.
